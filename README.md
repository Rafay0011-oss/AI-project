# Disease Detection System

## Objective

The objective of this project is to develop an AI-based Disease Detection System that predicts or identifies diseases using machine learning techniques based on the provided input data.

## Features

* Data preprocessing and cleaning
* Machine learning model training
* Disease prediction
* Model evaluation and performance analysis
* Interactive implementation in Jupyter Notebook

## Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## Project Structure

* `Disease_Detection_System.ipynb` – Contains the complete source code, data preprocessing, model training, and prediction.

## How to Run

1. Install Python and Jupyter Notebook.
2. Install the required libraries:

   * Pandas
   * NumPy
   * Scikit-learn
   * Matplotlib
3. Open the `.ipynb` file in Jupyter Notebook.
4. Run all cells in sequence.
